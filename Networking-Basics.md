# Networking Basics

## What is an IP Address?

An IP address is a unique identifier assigned to a device on a network.

Example:
192.168.1.1

## Types of IP Address

- Public IP
- Private IP

## What is a MAC Address?

MAC address is a hardware address assigned to a network interface.
