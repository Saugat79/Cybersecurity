# Networking Basics

## What is an IP Address?

An IP address is a unique identifier assigned to a device on a network.

Example:
192.168.1.1

## Types of IP Address

- Public IP
- Private IP

## What is a MAC Address?

MAC address is a hardware address assigned to a network interface.

# Network Devices Overview

## Introduction

Network devices are hardware components used to connect computers and manage communication within a network.

Understanding these devices is important for networking and cybersecurity.

---

## Hub

A hub is a basic networking device that connects multiple computers in a network.

- Works at OSI Layer 1 (Physical Layer)
- Broadcasts data to all devices
- No intelligence or filtering

---

## Switch

A switch connects multiple devices within a LAN.

- Works at OSI Layer 2 (Data Link Layer)
- Uses MAC addresses
- Sends data only to the intended device

---

## Router

A router connects different networks together.

- Works at OSI Layer 3 (Network Layer)
- Uses IP addresses
- Forwards packets between networks

---

## Repeater

A repeater regenerates signals to extend the distance of a network.

- Works at OSI Layer 1
- Amplifies and retransmits signals

---

## Bridge

A bridge connects two LAN segments and helps reduce network traffic.

- Works at OSI Layer 2
- Uses MAC addresses
- Filters traffic between network segments
